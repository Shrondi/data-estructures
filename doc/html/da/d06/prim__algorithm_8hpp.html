template<class T >
float prim_algorithm ( Graph< T, float > & g,
std::vector< typename Graph< T, float >::EdgeRef > & mst )

The Prim's algorithm. The Prim's algorithm create the minimum spanning tree on an undirected weighted graph.

  • [in] g is the graph.
  • [out] mst is the list of edges of g that forms the minium spanning tree.
    Returns
    the total weight of the minimum spanning tree.
    Precondition
    g.is_undirected()
    g.hash_current()
    Exceptions
    std::runtime_error("Itis an unconnected graph.") if the input graph is an unconnected one.
