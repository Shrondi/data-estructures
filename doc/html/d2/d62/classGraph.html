Detailed Description

template<class T, class E>
class Graph< T, E >

Models the Graph ADT.

Template Parameters
Tis the vertex's data type item. It is assumed that T type defines a type T::key_t and a method 'T::key_t T::key() const' return the key.
Eis the edge's data type item.
Invariant
!is_empty() || size()==0
!is_full() || size()>=capacity()
