Create a level of the kdtree object.

Create a binary tree in a recursive way splitting each dimension into two parts using the median as pivot and calling recursively for each part. The pivot will be the root of the binary tree.

Parameters
beginis the an iterator at the begin of patterns seq.
endis the an iterator at the end of patterns seq.
levelis the current level of this subtree.
kis the number of dimensions of the pattern's space.
Returns
a ref to the binary tree created of this level.
