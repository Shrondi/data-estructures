float kruskal_algorithm ( Graph< T, float > & g,
std::vector< typename Graph< T, float >::EdgeRef > & mst )

The Kruskal's algorithm. Compute the minimum spanning tree.

  • [in] g is the graph.
  • [out] mst is the list of edges of g that forms the minium spanning tree.
    Returns
    the total weight of the minimum spanning tree.
    Precondition
    g is an undirected connected graph.
    Exceptions
    std::runtime_error("Itis a unconnected graph.") if the input graph is an unconnected one.
