Detailed Description

Models a node of a Trie.

The path from the root node to this node represents a prefix and the this node represents a key equal to this prefix and all the following possible next prefixes where a new symbol was added.

For example: if we have the keys '123', '1231', '1232, '1233' a node will represent the key '123' and the prefix '123' with three child nodes for symbols '1', '2' and '3' respectively.
